# @ianwalter/eslint-config
> Ian's [ESLint][eslintUrl] configuration based on [Standard][standardUrl]

[![npm page][npmImage]][npmUrl]

## Installation

```console
yarn add @ianwalter/eslint-config --dev
```

## Usage

In `package.json`:

```js
  "eslintConfig": {
    "root": true,
    "extends": [
      "@ianwalter"
    ]
  }
```
